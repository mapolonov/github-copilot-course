### Типові питання учасників

**"Copilot бачить мій код в хмарі?"**  
Так, ваш код відправляється в GitHub/Microsoft для генерації. У Business/Enterprise — ваші дані не використовуються для навчання. Перевірте політику вашої компанії для sensitive проєктів.

**"Що буде з нашими робочими місцями?"**  
Copilot прибирає рутину, не досвід. Senior розробники, постановники задач, архітектори — стали потрібні ще більше. Цінність переміщується від "написати код" до "сформулювати правильно".

**"Як його використовувати для legacy VB.NET / C++ коду?"**  
Copilot підтримує їх, але гірше. Використовуйте Chat для пояснення та документації. Для migration запити працюють добре.

**"Ми не можемо використовувати GitHub — у нас Azure DevOps / GitLab"**  
GitHub Copilot працює через розширення IDE, незалежно від Git хостингу. Тільки Workspace вимагає GitHub. Для VS та VS Code — працює з будь-яким SCM.
